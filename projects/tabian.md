## 経営者向け動画プラットフォームの開発

**テックリード**

経営者向け動画プラットフォームの新規開発プロジェクトにテックリードとして参画。技術選定、コードレビュー、API呼び出しやキャッシュ戦略等の共通設計、インフラ基盤構築を担当した。

エンタープライズ向けアプリケーションのため、セキュリティ要件とコード品質に重点を置いて開発を進めた。3属性のユーザー（ユーザー / 動画投稿者 / 管理者）それぞれの画面を構築し、動画コンテンツのストリーミング閲覧機能の設計・実装も行った。

**技術**: TypeScript / Nx (モノレポ) / Nuxt.js 3 / Vuetify / Pinia / AWS Amplify / Lambda / DynamoDB / S3 / Cognito

## 求人応募を収集する基幹システムの開発

**テックリード**

## NPO法人 既存業務リプレイスシステムの開発

**単独開発**

NPO法人における既存業務のリプレイスシステムを単独で開発。顧客MTGから要件定義、設計、開発、テストまで担当した。

システム維持費用を極力抑えるため、GASをインフラ基盤、スプレッドシートをDBとする構成を採用。GASで対応できない要件はFirebaseにホスティングしたWebアプリケーションで対応した。

**技術**: TypeScript / Clasp (GAS) / GCP / Firebase / Firebase Authentication / Sheets API / Drive API
