## QOHS Home — 予約・顧客管理システム + スマホアプリ

**開発責任者 / 2020年12月 - 2022年1月**

新規事業の予約システムの開発に開発責任者として参画。入社時に新規事業のプロダクトづくりに携わることになり、ビジネスモデルの設計から顧客導線の設計、システム設計、開発、リリース、保守まで一通り対応した。

インフラの立ち上げからバックエンドサーバー、スマホアプリ、管理画面まで開発しリリース。顧客ヒアリングを通じたアプリケーション改善も実施した。2021年6月以降は新メンバーと共に運用・保守・機能追加を担当。

**プロジェクト要件:**
- 整体師派遣事業の予約システム
- 整体師 / 顧客 / 管理者 の3種ログイン
- シフトベースの予約受付機能
- ポイント機能・サブスク機能

**技術**: AWS EC2 / Laravel / PHP / Vue2 / JavaScript / Ionic / Adobe XD

**単独開発 / 2020年4月 - 2020年11月**

**技術**: PHP / Linux / Adobe XD
