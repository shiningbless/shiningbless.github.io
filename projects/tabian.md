## GASを使用した既存業務のリプレイスシステム

**役割**: 単独開発

NPO法人の既存業務リプレイス。顧客MTGから要件定義、設計、開発、テストまで1人で担当。

**技術スタック**: TypeScript / Clasp (GAS) / GCP / Firebase / Firebase Authentication / Sheets API / Drive API
